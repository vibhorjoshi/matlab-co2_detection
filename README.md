Markdown# Hyperspectral CO2 Detection using Cluster-Based Target Matched Filter (CTMF)

This repository provides a MATLAB-based framework for detecting localized $CO_2$ gas plumes in hyperspectral imagery (HSI). Standard global matched filters often struggle with heterogeneous backgrounds (e.g., urban areas, mixed vegetation), resulting in high false-alarm rates. This project implements a **Cluster-based Target Matched Filter (CTMF)** that groups pixels into localized backgrounds, computing precise covariance matrices to significantly improve the Signal-to-Clutter Ratio (SCR).

🧠 Algorithms Explained1. Cluster-based Target Matched Filter (CTMF)The CTMF algorithm (compute_ctmf.m) enhances the traditional Matched Filter by adapting to local background clutter rather than assuming a single global background distribution.Flattening & Clustering: The 3D hyperspectral cube is flattened into a 2D matrix ($N \times B$, where $N$ is pixels and $B$ is bands). K-Means clustering partitions the pixels into $K$ distinct background classes.Localized Covariance ($\Sigma$): For each cluster, the algorithm computes a regularized sample covariance matrix:$$\Sigma = \frac{X_c^T X_c}{n_k - 1} + \lambda I$$where $X_c$ is the mean-centered data for the cluster, $n_k$ is the number of pixels, and $\lambda$ is a small regularization factor ($10^{-6}$) to ensure the matrix is invertible.Matched Filter Application: The filter vector $v$ is calculated by solving $\Sigma v = d$, where $d$ is the known $CO_2$ target signature. The final pixel score is computed as:$$Score_k = \frac{X_c v}{\sqrt{d^T v}}$$This isolates the $CO_2$ signature while fiercely suppressing localized background clutter.2. RGB Composite GenerationThe get_rgb_composite.m function creates human-readable visualizations from the hyperspectral cube.Band Extraction: It targets $660$ nm (Red), $550$ nm (Green), and $470$ nm (Blue) wavelengths.Window Averaging: Instead of picking a single noisy band, it averages bands within a $\pm 10$ nm window of the target.Contrast Stretching: Applies a robust 2nd-to-98th percentile stretch per channel to remove outliers and enhance visual contrast, returning a clean uint8 RGB image.📊 Visualizations and FiguresThe pipeline produces two primary visualizations: the RGB composite of the target area, and the heat map of the $CO_2$ detection scores.Observation / Detection Map: The brighter/warmer colors (yellow-orange) indicate high matched filter scores, confirming the presence and concentration morphology of the $CO_2$ plume.RGB Composite Background: Demonstrates the underlying terrain (e.g., industrial stacks). Mixing grass, concrete, water, and dirt into one covariance matrix flattens out critical local statistical anomalies, causing high-reflectance background clutter to trigger false $CO_2$ alarms.The CTMF Solution: By clustering the image first, CTMF calculates a background covariance specific to concrete, a separate one for grass, etc. When looking for the $CO_2$ signature $d$, the filter only has to compete against the localized background clutter, not the entire scene, resulting in a vastly superior Signal-to-Clutter Ratio (SCR) and cleaner detection maps.
